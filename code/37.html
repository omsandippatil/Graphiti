▶ Algorithm Theory & Explanation
Algorithm Overview:
This algorithm iterates through each character in the input string and uses the indexOf method to check if the character exists in a predefined vowels string. If indexOf returns -1, the character is not a vowel, otherwise it is.

Key Concepts:
• indexOf() returns the first index where the character is found, or -1 if not found
• We define vowels as "aeiouAEIOU" to handle both cases
• Characters are classified as either vowels or consonants (ignoring non-alphabetic)

Time Complexity: O(n) where n is the length of the string
Space Complexity: O(1) for the variables used
▶ Edge Cases & Boundary Conditions
Edge Cases:
• Empty string: returns 0 vowels, 0 consonants
• String with only vowels: consonantCount remains 0
• String with only consonants: vowelCount remains 0
• Mixed case letters: handled by including both cases in vowels string
• Non-alphabetic characters: treated as consonants in this implementation

Boundary Conditions:
• First character (i = 0)
• Last character (i = string.length - 1)
• Single character string
▶ Implementation Tricks & Optimizations
IndexOf Technique:
• Using indexOf is more readable than multiple if conditions
• Returns -1 when character not found, any other value means found
• Single comparison: vowels.indexOf(char) !== -1

Alternative Approaches:
• Using includes(): vowels.includes(char)
• Using Set for O(1) lookup: vowelSet.has(char)
• Using regular expressions: /[aeiouAEIOU]/.test(char)

Performance Notes:
• indexOf has O(k) complexity where k is vowels string length (10)
• For large strings, Set-based approach would be faster
• Current approach is fine for typical string lengths
