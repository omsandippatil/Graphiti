Algorithm Theory

Two-Pointer Technique

Uses two pointers to traverse both arrays simultaneously, comparing elements and selecting the smaller one for the result array.

Time Complexity

O(m + n) where m and n are the lengths of the input arrays.

Space Complexity

O(m + n) for the result array.

Key Points

  • Compare current elements from both arrays
  • Add smaller element to result and advance its pointer
  • Handle remaining elements after one array is exhausted
