Problem: Find Duplicates in Array

Input: [4,3,2,7,8,2,3,1]

Output: Find all duplicates → [2,3]

Algorithm Explanation:

This algorithm finds duplicates in an array using the "Cyclic Sort" approach. It modifies the input array by placing each number at its correct index (number n goes to index n-1). During this process, if we find a number already at its correct position, it's a duplicate.

Key Insights:

  • Array contains numbers from 1 to n, some appearing twice
  • Each number should be at index (number - 1)
  • When we can't place a number due to collision, it's a duplicate
  • Time: O(n), Space: O(1) if we don't count result array

Edge Cases:

  • All numbers are duplicates
  • No duplicates exist
  • Single element array
  • Array with multiple duplicates of same number

Algorithm Steps:

  1. Iterate through each position in array
  2. For current number, check if it's at correct position
  3. If not, try to swap it to correct position
  4. If target position already has correct number, current is duplicate
  5. Add duplicates to result and continue
