▼ Algorithm Theory & Explanation

Dynamic Programming Approach

The LIS problem uses dynamic programming where dp[i] represents the length of the longest increasing subsequence ending at index i.

For each element, we check all previous elements. If a previous element is smaller, we can extend its subsequence.

Time complexity: O(n²), Space complexity: O(n)

▼ Algorithm Tricks & Optimizations

• Initialize all dp values to 1 (each element forms a subsequence of length 1)

• For each position i, check all positions j < i

• If nums[j] < nums[i], then dp[i] = max(dp[i], dp[j] + 1)

• The answer is the maximum value in the dp array

▼ Edge Cases & Boundary Logic

• Empty array: return 0

• Single element: return 1

• All elements equal: return 1

• Strictly decreasing: return 1

• Already sorted: return array length
