Theory & Algorithm Explanation
Kadane's Algorithm

Dynamic programming approach to find maximum sum contiguous subarray in O(n) time.

Core Idea: At each position, decide whether to extend the current subarray or start a new one.

Algorithm:
1. Initialize maxSoFar and maxEndingHere to first element
2. For each remaining element:
  - maxEndingHere = max(current element, maxEndingHere + current element)
  - maxSoFar = max(maxSoFar, maxEndingHere)

Key Insight: If maxEndingHere becomes negative, start fresh from current position.
Tricks & Optimizations
Memory Optimization: Only need two variables, not entire DP array

Early Termination: Can stop if all remaining elements are negative

Variant - Track Indices: Keep start/end pointers to return actual subarray

Handle All Negatives: Return maximum single element if all negative
Edge Cases & Boundary Logic
Empty Array: Return 0 or handle as special case

Single Element: Return that element

All Negative: Return maximum single element

All Positive: Return sum of entire array

Mixed Signs: Core algorithm handles naturally
