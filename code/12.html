Theory & Explanation ▼
Boyer-Moore Majority Vote Algorithm

Uses O(1) space and O(n) time. Maintains a candidate and count.

Key Insight: If we cancel out different elements, the majority element will remain.

Algorithm:
1. If count is 0, set current element as candidate
2. If current element equals candidate, increment count
3. Otherwise, decrement count

Why it works: The majority element appears more than n/2 times, so even after canceling with all other elements, it will have a positive count.
