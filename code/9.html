Algorithm: Three-Step Reversal Method

1. Reverse entire array
2. Reverse first k elements
3. Reverse remaining elements

Time Complexity: O(n)
Space Complexity: O(1)

Edge Cases:
- k > n: Use k % n
- k = 0: No rotation needed
- Empty array: Return as is

Why It Works:
Reversing transforms the relative positions systematically to achieve the desired rotation pattern.
