for (int i = 0; i < 5; i++) { }
Input: for(int i=0; i<3; i++) { System.out.println(i); } Output: Prints 0, 1, 2
Executes a block of code repeatedly while a condition is true, with initialization and increment.
Extended Notes:
• Initialization runs once at the start
• Condition is checked before each iteration
• Update expression runs after each iteration
• Any part can be omitted (infinite loop possible)

Edge Cases:
• Enhanced for-each loop for collections
• Multiple variables can be declared
• Break and continue alter flow control
i = 0
i < 3
i++
Iteration 0: i=0
Iteration 1: i=1
Iteration 2: i=2
