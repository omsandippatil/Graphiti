Extended Notes:

Operator Precedence: * / % have higher precedence than + -
Type Coercion: JavaScript automatically converts types in many operations
Edge Cases: Division by zero returns Infinity, modulo by zero returns NaN
Bitwise Operations: Work on 32-bit signed integers
Short-circuit Evaluation: && and || don't always evaluate both operands
Assignment Chaining: a = b = c assigns c to both a and b
Comparison Gotchas: === checks type and value, == performs type coercion
