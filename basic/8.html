ArrayList Implementation: Backed by resizable array. Default capacity 10, grows by 50% when full. O(1) amortized add, O(n) worst case.

LinkedList Implementation: Doubly-linked list. O(1) add/remove at ends, O(n) for middle operations.

HashMap Implementation: Hash table with separate chaining. O(1) average case, O(n) worst case. Load factor 0.75.

HashSet vs TreeSet: HashSet uses HashMap internally. TreeSet uses Red-Black tree, maintains sorted order.

Collections Utility: Static methods for common operations. Uses optimized algorithms like TimSort for sorting.
