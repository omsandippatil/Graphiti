Scanner Input Methods
Scanner scanner = new Scanner(System.in);
scanner.next()
Example:
Input: "Hello World"
String word = scanner.next();
Output: word = "Hello"
Reads the next token (word) from input, stopping at whitespace.
Notes:
• next() reads until whitespace (space, tab, newline)
• nextLine() reads entire line including spaces
• nextInt() parses integer, throws exception if not valid
• nextDouble() parses decimal number
• Always close scanner when done: scanner.close()
• Can read from files: new Scanner(new File("file.txt"))
